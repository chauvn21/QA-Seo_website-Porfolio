# CAMEL BEER WEBSITE

## Giới thiệu

Đây là dự án Portfolio Manual Testing được xây dựng nhằm mô phỏng quy trình kiểm thử hoàn chỉnh cho một Website giới thiệu sản phẩm của Camel Beer.

Dự án bao gồm tài liệu BA, Test Plan và Test Case được thiết kế theo quy trình thực tế của một Manual QA.
